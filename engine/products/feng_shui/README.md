# 풍수·인테리어 가이드 (feng_shui)

가격: 69,000원 / 분량: 20,000~70,000자 / cover 동물 있음

## 필요한 추가 입력 (구매 시 폼)
- 현재 거주 형태 (아파트/단독/오피스텔/원룸)
- 현재 집 향(좌향) — 동/서/남/북/모름
- 본인 침실 위치(현관 기준 방향) — 선택, 모름 허용
- 거주 지역 (시·도) — 방위 풀이 보정용
- 주 사용 공간 (재택근무 여부, 작업방 여부)

## 필수 표 (PRODUCT_SPECS.required_tables)
- cover (풍수·인테리어 전용 cover)
- 명식표
- 인적사항표
- 사주기본표
- 오행인테리어비교표
- 용신가이드카드
- 지지계절방위표

## 필수 본문 (PRODUCT_SPECS.required_data)
- ch03 (오행 풀이 — 풍수 기반)
- ch_interior (인테리어 가이드 신규 챕터)

## 풍수 매칭 규칙 (구현 가이드)
- 용신 오행 → 추천 색상/소재/방위/식물
  - 木: 동/동남, 녹색, 목재, 식물·키 큰 화분
  - 火: 남, 빨강·주황, 조명, 삼각형/뾰족한 형태
  - 土: 중앙·남서·북동, 노랑·황토, 도자기/세라믹, 사각형
  - 金: 서/북서, 흰색·금색, 금속, 원형/구형
  - 水: 북, 검정·파랑, 유리, 곡선/물방울
- 기신 오행은 해당 색/소재 최소화 권장
- 일간 좌향 vs 현재 좌향 차이 풀이
