# 택일 (결혼·이사·개업·계약) (auspicious_day)

가격: 59,000원 / 분량: 15,000~60,000자 / cover 동물 있음

## 필요한 추가 입력 (구매 시 폼)
- 행사 종류 (필수): 결혼/이사/개업/계약/기타
- 후보 기간 (시작일~종료일) — 1~6개월 범위
- 동반자 사주 (선택, 결혼/계약 시 권장) — 생년월일·성별
- 방위 제약 (이사 시) — 출발지/도착지 시·도
- 우선순위 (안정/속도/재물/관계)

## 필수 표 (PRODUCT_SPECS.required_tables)
- cover (택일 전용 cover)
- 명식표
- 인적사항표
- 사주기본표
- 운세달력_전체 (후보 기간 전체)
- 운세달력_표지
- 용신가이드카드

## 필수 본문 (PRODUCT_SPECS.required_data)
- ch03 (오행)
- ch08 (대운/세운)
- ch09 (택일 본문 신규 — 행사별 길일 추천)

## 택일 점수 규칙 (구현 가이드)
- 일진(日干支) 5신 점수: 용신+30, 희신+18, 기신-25, 구신-12, 한신+5
- 행사별 가중치
  - 결혼: 합 가중↑, 충/형 가중↑↑(부정)
  - 이사: 역마/방위, 일지 충 회피
  - 개업: 일간 통근, 재성 길일
  - 계약: 정관·정인 길일, 충/형 회피
- 동반자 사주 입력 시: 두 사람 일주 합/충 검사
- Top 3 후보일 + 시간(時) 추천

## 참고
- 운세달력은 기존 generate_운세달력.js를 후보 기간 모드로 호출
- 5신 분류는 saju_calc 결과 그대로 사용 (단일소스 원칙)
- 폰트는 engine/fonts 폴더 폰트만 사용
